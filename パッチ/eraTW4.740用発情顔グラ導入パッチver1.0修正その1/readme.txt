TW用の発情時顔グラ追加パッチです。
ネームドキャラはムラムラすると顔画像もムラムラします（椛とお燐はすでに他の方が作成したえっちな特殊照れグラが入っているので除外）
時間と体力的な都合で服装などのバリエーションが別の画像ファイルにあるキャラ（霊夢とか咲夜とかヘカとか）など一部非対応。

下記のファイルを全てTWフォルダ内同名ファイルに上書きして下さい。

resources
・顔画像ファイル（たくさん）
※バックアップ推奨。万が一の為、元の画像に戻せる状態にしておいて下さい。
・顔.csv（発情顔のリストの追加）

ERB
→ステータス表示関連
・IMAGE.ERB（発情顔が表示される条件の追加）
※こちらもバックアップ推奨。詳細は下記。

────────────────────

↓↓↓↓↓必読↓↓↓↓↓

差し替えた画像に問題や違和感があったらバックアップの元の画像に差し戻して下さい。
その場合、除外したキャラの発情顔差分は非対応になります。
また、除外した場合はお手数ですが、IMAGE.ERBの358行目のcase欄から該当するキャラの番号を消して下さい。
消さないと該当キャラが発情した時に「照れ」グラが表示されず、ムラムラしてるのにすました顔をされます。
もしくは本パッチをさっぱり忘れて全ての顔画像ファイルとIMAGE.ERBをバックアップのものに戻して下さい。
csvはそのままでもたぶんだいじょうぶです。

また、繰り返しになりますが元画像加工者様より申し立てがあった場合は削除対応を行います。
