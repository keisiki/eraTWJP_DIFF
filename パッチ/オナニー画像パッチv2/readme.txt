# eraTW4.750用オナニー画像パッチv2
## 概要
夜オナニーの報告に顔グラが追加されます

## 使い方
解凍したファイルを素のeraTW4.750に当ててください(上書き保存すればok)

## 使用に関して

使用した立ち絵作者様：kaoru、dairi&はるか
画像の著作権はkaoru氏、dairi&はるか氏に帰属します。
作者様への問い合わせはご遠慮下さい。

作者様のサイト（利用規約）
http://gensoukyou.1000.tv/
https://goo.gl/4KDHgg

本パッチ内の画像ファイルはeraおよびeraろだ、派生系アップローダー以外へのアップロードを禁止します。

また、形式の都合により過去に画像を加工された方が作成した「全て」の画像に上書きする形で干渉しています。
画像加工者様より申し立てがあった場合は削除対応を検討させて頂きますのであらかじめご了承下さい。


ソースコードの改変はeraTWのライセンスと同様でお願いします。

このパッチ自体のバージョンアップ等は大歓迎です。ご自由にソースを改変してバージョンアップとしていただいてかまいません。
