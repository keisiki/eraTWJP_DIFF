■01_放尿解放
放尿コマンドとは別に、おもらし癖や利尿剤飲ませておくと絶頂時に漏らしてくれます。
利尿剤の効果は時間経過とは別に、放尿で効果終了の他、絶頂漏らしでも効果時間が短くなります。
とりあえず試作開始版で解放。基本上書きでなんとかなるはず。
「虫捕りパッチ」入れてる人は「虫取り用追いパッチ」を追加で書き換えればいけるはず。

追加でこの辺いじったらどうだろうリスト
・えーりんの診察か神頼みで「おもらし癖」削除
・着衣放尿
・放尿快楽経験とか追加したけど使ってない
・布団が色々大変な事になってそうなのでその辺り
・でも朝のお布団干しとかはＴＷだと表現難しいだろうなぁ。
