まとめパッチ修正点：

●　立ち絵システムをほぼ書き直し

    たくさんのダミー画像重ねて重くなる仕様を改善
    これでだいぶ軽くなるはずです

    追加エフェクトで行ずれる現象をなくしました

    大妖精や映姫など大き目サイズの画像導入するともに
    キャラ画像をマルチサイズ対応させ、好きな画像サイズを設定できるようしました
    画像サイズが違うことでエフェクトや照れがずれる問題も解決しました

    キャラごとに簡単に画像を追加と管理するために新しい関数Chara_EPとChara_DPを作成
    EP=Emotion exPansoin DP = Dress exPansionの略です。
    
    そして、顔絵付きメッセージに合わせて、キャラ画像が簡単設定できる関数SET_CHARAなどを作成
    今後のキャラ追加管理ももっとやりやすいように、モブ子のファイルを独立させました。

    条件確認系やPRINT_TEXT,PRINT_FACEなどをADD_FUNCTION.ERBに、
    画像作成関係をIMAGE2.ERBに、
    キャラごとの画像差分設定はchara_imagedata.erbに移行しました
    これでキャラ画像追加する度、システム面のコード影響せずに追加できます
    そしてシステム面で更新や改造する時も対照しやすく、マージしやすくなると思います。


●　キャラ紹介ページ調整
    キャラ紹介ページがどんどん長くなるため、
    所持アイテムを独立させて、
    新しいバージョンに合わせて色々改善しました。
    SHOP画面や、日常モードでは　いつでも【所持アイテム確認】のコマンドで確認できます


●　日記と顔つきメッセージ
    もう少し使いやすいようにコード面改善
    そして人がいない私室に入った場合、私室の主人じゃないキャラと日記読む場合なども対応
    そしてモブ子店員パッチに合わせて少し調整。モブ子店員パッチ導入してもしなくても大丈夫なはずです
    ついでに、既読・未読で演出変化するPRINT_TEXT関数を導入
    これを応用して、他の口上に使うこともできます。
    書き方も簡単です

    CALL PRINT_TEXT(代入する判定条件あるいは数値,"メッセージ内容/L/100行まで反応できるおー")
    例：
    一行にまとまる書き方

    CALL PRINT_TEXT(MODE,@"○月×日 /L/今日、 /L/外界から来たあの人に出会ったようです。[W]/L/ ただ●●を探すことでいっぱいになって、 /L/   /L/ ちゃんと…挨拶できなかった…[W]/L/……いや、わたしが…挨拶もできやしないかもしれないです[W]/L/外界の人とはいえ、やはり人間です /L/なんだか怖いですぅ[W]/L/あう、もう二度と合わせないで…[WW]")

    改行できる書き方

{
    CALL PRINT_TEXT(MODE,@"○月×日 /L/ 
    今日、 /L/ 
    外界から来たあの人に出会ったようです。[W]/L/ 
    ただ●●を探すことでいっぱいになって、 /L/   /L/ 
    ちゃんと…挨拶できなかった…[W]/L/ 
    ……いや、わたしが…挨拶もできやしないかもしれないです[W]/L/ 
    外界の人とはいえ、やはり人間です /L/ 
    なんだか怖いですぅ[W]/L/ 
    あう、もう二度と合わせないで…[WW]")
}

    /L/ で改行
    [W]　MODE（未読、あるいは何等かのフラグ）がTRUEの時にWAIT,FALSEの時に待たせず次の行をPRINT
    [WW] で強制WAIT

    仕様が少し変わったため、映姫の日記でエラー出そうなので直しておきました

●　BUGFIXとコード改善

    もっと古いバージョンやキャラ追加された後アップデートすると
    キャラデータがずれたり、画像が間違ったりする問題を徹底解決

    新バージョンで種族がちゃんと追記として表記させるようにしました

    家具が完成されない問題を解決、そして家具強化した後休憩や昼寝するとき発生するエラーを解決
    木材商と家具屋をひとまず使えるようにしました

    日常モードのINFOでＴＳＰが二回プリントするBUG修正
    たまに起きる０による除算され～というエラーを徹底解決

    携帯中のキャラがいて同じ場所にキャラが表示キャラ以上にいて、
    携帯中キャラが表示外の時に起きるTARGETが-1というエラーを徹底解決

    調合するとき、単一素材レシピで消耗素材の数がおかしくなるBUG修正
    露店のコードを改善、売れるアイテムに統一

    アナル処女喪失時のフラグや地の文を補完した
