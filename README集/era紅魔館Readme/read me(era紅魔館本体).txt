遊び方
	1.外付け型オープニングを開いてオープニングを作成する
	2.作成したオープニングをじっくりと読む
	3.おもむろに同梱のEMUERAを起動する

バリアントの特徴
	・紅魔館を舞台としたADVゲームを目指してみた
		館見取り図を拡張
	・ベースはeracustom
	・MASTERのカスタムが可能
		男、女、ふたなり、ショタ、ロリお好きな属性でどうぞ
		男の娘、男装少女にも対応
	・TARGETが常に変動する
		MASTERと自動でうろうろする住人が同じ部屋にいるときにTARGET:1〜が設定され、TARGET:1の住人がTARGETとなる
		同じ部屋に複数の住人がいるときはCFLAG:310により設定される館内での地位の順でTARGET:1〜が設定される
		処理の都合上TARGETの初期値は0
	・住人と会話したりスキンシップしたり、部屋の掃除をしたりできる
		TARGETとの親密が上がるとコマンドごとに積極的に行動してくるようになる
			カウンターは今のところSOURCE等の処理は行われず状況描写だけ
		理性とムード、刻印、欲望、欲情、好意なんかも考慮するように
		着衣セクハラを追加
			着衣状態での露骨なセクハラをされます
	・能力の上昇は自動で行われる
	・現在のところ館を清潔に保つことでMONEYが得られる。
		MONEYの使い道はアイテムの購入の他、館内での地位の上昇などを予定している
		部屋に住人がいるとその部屋が時間経過により汚れていく
		清掃技能を持つ住人なら勝手に掃除してくれる
	・各種技能は経験により上昇する
	・時間
		TIMEで管理している時間は分刻みで、コマンドごとに一定の時間が経過し、1440分経過すると１日繰り上がる
		お茶を入れるなど特定のコマンドは一定時間経たないと実行できないようになっている
	・メイド妖精詰所
		住人の居場所、部屋の清潔度を調べることができる
	・日替わりパンツシステム
		よごれている部屋を掃除すると脱ぎっぱなしのパンツをくすねることが出来るように
		パンツはSHOP画面で閲覧、売却(同種のパンツを2枚以上持っているとき一律100$)が可能
	・感情ゲージ
		ムード、理性、怒りの三種類
		取得SOURCE、時間経過で上下する
		怒りゲージが限界に達すると怒り状態になり、コマンドが制限され、また怒りゲージが減少しにくくなる
		ムード、理性はMASTERとの馴れ合い強度、取得SOURCEに影響する
	・うふふ
		MASTERの馴れ合い強度1以上でセクハラが、2以上で押し倒すことが可能になる
		服を脱がせる[820]で全裸にする
			脱衣はもっと充実される予定
		うふふモードは 逢引終了[999] で解除できる
		うふふ中に3人目が同室するとうふふモードは解除される
		うふふ中TARGETの体力が500を切ると強制解除され、TARGETはしばらく自室で就寝する
		二人以上と同室しているとき両者の馴れ合い度が高いとまとめて押し倒すことが出来るように
		性交系のソースは入るがバイブなどのソースはまだ入らない
	・反発刻印
		ムード低い、理性高い状態ではマイナスSOURCEに補正がかかり反発刻印が付きやすい
		反発刻印がつくと数日間不機嫌になるが日数経過で不機嫌は解消し、反発刻印は除去される
			反発1では次の日には機嫌が直り反発刻印は除去されるが、2,3は数日かかる
		不機嫌状態では常に怒りゲージがMAX
			コマンドをほとんど実行できなくなる
			うふふ中に反発刻印がつくとうふふフラグが解除される
		反発刻印は何度でも取得するが反発刻印によるABL低下は一回だけ
	・イきそう
		射精箇所選択の代わりとして導入
		イキそうになるまでは射精しない
		勃起度を導入
			非勃起状態なら一部のコマンドが制限される
	・コマンドの整備
		コマンドを実行すると@COM999(ダミー)がよばれ@SOURCE_CHECKにより@COMXを呼び出す
		実行判定を@CAN_COMXに分離した
		コマンドごとに体位がTEQUIP:101〜に設定され、時間をさかのぼって何をしていたかがわかる
			クンニ→アナル愛撫ではクンニ状態は引き継がれるが、クンニ→キスならば引き継がれない
				TEQUIP:101〜はTEQUIP:111〜に1ターン引き継がれる
				パイズリとフェラは本来排他的だが、TEQUIP:111〜をみることでパイズリとフェラを連続的に行なったことを判定できる
			クンニ→アナル愛撫→６９(フェラチオ)というようなことが出来る
			TARGETごとに設定されるのでうまく動かせば3P,4P,〜にも対応できるはず
				とりあえずダブルフェラを実装
			バグ取り用に@SHOW_STATUSで今何をやっているか簡単な表示が出力される
				鬱陶しいと思う人はINFO.ERB 3,4行目の
					SIF TFLAG:102
						CALL SHOW_TOUCH
				を削除するかコメントアウトしてください
		派生コマンドはSELECTCOMでなくTFLAG:50で管理する
			SELECTCOMに干渉しないのでむやみに派生を増やしてテンプレを圧迫することもなくなる
			派生コマンドは、@COMABLE501〜,@SCOM1〜,@CAN_COM1〜で管理する
			
		COMFX.ERBからPRINT系の命令を排除
			@COMX内でCALL COMXみたいな使い方が出来るようになった
				例：手淫フェラ(@SCOM5)ではまずフェラチオを呼び出して手淫経験、快感強度を上書きしている
	・SOURCEの整備
		CUPを用いてキャラごとに管理するように
		TARGETが省略されていた部分は別関数にしてARGを用いるように
		MASTERとTARGETに設定されているキャラについてはSOURCE計算が回るように
	・陥落素質
		好感度1000以上、奉仕快楽経験200以上で恋慕取得
		恋慕取得時に相手の部屋の合鍵をもらえる
		フランちゃんの場合は幽閉が解かれる
		恋慕の相手の部屋をMASTERの私室にしてそこで寝起きすることが出来る
		※NEW※
		各種の性技の熟練度が上がるように
		熟練度MAXで部分陥落素質もしくは技能素質を習得
		カウンター行動に影響
	・大浴場を設置
		時間経過により風呂への移動傾向が高くなる
		大浴場でうふふ状態になるとお風呂場プレイがつく
		それに伴い通常コマンドのお風呂場プレイを廃止
		セクハラ、逆セクハラするとガチエロになるので整備されるまでは会話とスキンシップ程度、押し倒す、連れ出すしか出来ない
		フランちゃんおふろではおしりのあなもきれいにするんだようふふ
			風呂にはいると汚れがリセットされ、肛門の汚れが落ちる
			時間経過で肛門にアナル汚れがつく
			汚れに腸内も追加、アナルセックス、アナル愛撫などでアナル汚れが肛門に付着
			汚れ無視がなくてもアナル舐めが出来るようになった
構想中の仕様
	・セクハラ
		着衣日常系のコマンドを増やしたい
		感情ゲージ的なもので許容量を設定する
	・逆セクハラ
		着衣逆セクハラを実装
		SOURCEがはいるように
			MASTERのSOURCEはTARGET側で管理
		脱衣セクハラをいくつか実装
			もうちょっと調整が必要
	・ウフフモード
		セクハラを繰り返す、逆セクハラで押し倒されると発生するように
		他の住人に見つかることがある
		納屋では見つからない
	・姉妹丼主従丼洋風中華丼
		PLAYRE-TARGET主体ではなくMASTER-TARGET:1 MASTER-TARGET:2…のようなシステムを目指す
	・侵入者
		たまに魔理沙が館に侵入しますが試験運用中です。特に害はないので生ぬるい目で見守ってください
	・テンプレ
		最大限詰め込むとテンプレサイズが膨大になり見ただけでやる気が奪われる気がする
			分割テンプレにより実質的には問題ないが全部埋めないといけないと思ってしまう作者がいそうな気がする
		テンプレは最小限のサイズに抑える
		小分岐集を別に作り、口上作者はそれを参考に分岐を作成する
		パイズリフェラ口上を書かなければパイズリ口上が出るようなギミックを仕込む
		※NEW※
		テンプレにライセンス的なものをつけた
			基本的に作者の意思が不明な場合は加筆改変は可能ということにします
			改変元を明らかにするなど最低限のマナーはここに明記しなくても分かっているはず
