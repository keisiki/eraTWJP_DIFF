・導入方法
-------------------------------------------------------------------------------------------
※注意
※最新環境のeraTW(現時点ではver4.821)が必須です。過去のバージョンでの動作は保証しません

eraTW/ERB/口上・メッセージ関連/個人口上/豊姫/
ERBフォルダをそのまま上書きすれば動作する筈です
起床前のメニューで[UPDATE]を押すことで変更可能な設定があります。確認してみてください
-------------------------------------------------------------------------------------------
